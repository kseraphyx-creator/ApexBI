How to Add Your Business Data
1
Open Excel, Tally, or Google Sheets
Export your monthly sales data from whatever software you use.
2
Create these 5 column headers exactly
First row must have: Month, Revenue, Expenses, Product, Region
3
Fill your data row by row
Revenue and Expenses in ₹ numbers only (no ₹ symbol). Month format: 2024-01
4
Save as CSV
Excel: File → Save As → CSV. Google Sheets: File → Download → CSV.
5
Upload below — dashboard updates instantly
Your data is saved automatically. You will NOT need to re-upload next time.
Example CSV Format
Month,Revenue,Expenses,Product,Region 2024-01,45000,28000,Lehenga,India 2024-01,12000,8000,Kurti,Delhi 2024-02,52000,31000,Lehenga,India 2024-03,48000,29000,Saree,Mumbai
